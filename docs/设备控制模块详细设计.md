## 2. 模块概述

### 2.1 职责

设备控制模块负责向米家设备发送控制指令，查询设备状态，处理控制结果反馈。

### 2.2 功能范围

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 开关控制 | 打开/关闭设备 | P0 |
| 亮度调节 | 调节灯光亮度 | P0 |
| 色温调节 | 调节灯光色温 | P1 |
| 颜色调节 | 调节灯光颜色 | P2 |
| 状态查询 | 获取设备当前状态 | P0 |
| 批量控制 | 同时控制多个设备 | P1 |
| 定时控制 | 定时执行控制指令 | P2 |

### 2.3 依赖关系

### 6.1 服务架构

```

│              Python Device Service                      │
│                 (FastAPI + python-miio)                 │
├─────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────┐ │
│  │ HTTP Server │  │ Device Pool │  │  Miio Client    │ │
│  │  (FastAPI)  │  │  (连接池)   │  │  (python-miio)  │ │
│  └─────────────┘  └─────────────┘  └─────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

### 6.2 核心代码结构

```python
# main.py
from fastapi import FastAPI
from miio import Device

app = FastAPI()

device_cache = {}

@app.post("/api/devices/{device_id}/control")
async def control_device(device_id: str, command: ControlCommand):
    """控制设备"""
    device = await get_device(device_id)
    result = device.send(command.method, command.args)
    return {"success": True, "result": result}

@app.get("/api/devices/{device_id}/status")
async def get_device_status(device_id: str):
    """获取设备状态"""
    device = await get_device(device_id)
    status = device.status()
    return status

@app.post("/api/batch/control")
async def batch_control(commands: list[ControlCommand]):
    """批量控制"""
    tasks = [control_device(cmd.deviceId, cmd) for cmd in commands]
    results = await asyncio.gather(*tasks, return_exceptions=True)
    return {"results": results}

async def get_device(device_id: str) -> Device:
    """获取设备实例（带缓存）"""
    if device_id not in device_cache:
        device_info = await load_device_info(device_id)
        device_cache[device_id] = Device(device_info.ip, device_info.token)
    return device_cache[device_id]
```

### 6.3 设备类型适配

```python
# adapters.py
from miio import Device, PhilipsBulb, Yeelight, ChuangmiPlug

DEVICE_ADAPTERS = {
    'philips.light.*': PhilipsBulb,
    'yeelink.light.*': Yeelight,
    'chuangmi.plug.*': ChuangmiPlug,
}

def get_device_adapter(model: str, ip: str, token: str) -> Device:
    """根据 model 获取适配器"""
    for pattern, adapter in DEVICE_ADAPTERS.items():
        if fnmatch(model, pattern):
            return adapter(ip, token)
    return Device(ip, token)  # 默认使用通用设备
```

## 7. 异常处理

### 7.1 异常类型

| 异常 | 触发条件 | 处理方式 |
|------|----------|----------|
| DeviceOffline | 设备离线 | 重试 3 次，标记离线状态 |
| InvalidToken | Token 无效 | 提示重新配置 |
| TimeoutError | 超时 | 重试 2 次，返回超时错误 |
| DeviceNotSupported | 不支持的方法 | 返回错误，提示设备不支持 |

### 7.3 重试机制

```typescript
async function sendCommandWithRetry(
  deviceId: string, 
  method: string, 
  args: any[],
  maxRetries = 3
): Promise<CommandResult> {
  for (let i = 0; i < maxRetries; i++) {
    try {
      const result = await sendCommand(deviceId, method, args);
      return { success: true, deviceId, method, timestamp: new Date() };
    } catch (error) {
      if (i === maxRetries - 1) {
        return { 
          success: false, 
          deviceId, 
          method, 
          error: error.message,
          timestamp: new Date()
        };
      }
      // 等待后重试
      await sleep(1000 * (i + 1));
    }
  }
}
```

        self.pool = {}

### 8.2 命令队列

```typescript
// 命令队列（防止并发冲突）
class CommandQueue {
  private queue: ControlCommand[] = [];
  private processing = false;

  async enqueue(command: ControlCommand): Promise<CommandResult> {
    return new Promise((resolve) => {
      command.callback = resolve;
      this.queue.push(command);
      this.process();
    });
  }

  private async process() {
    if (this.processing || this.queue.length === 0) return;
    this.processing = true;

    while (this.queue.length > 0) {
      const command = this.queue.shift()!;
      const result = await executeCommand(command);
      command.callback?.(result);
    }

    this.processing = false;
  }
}
```
