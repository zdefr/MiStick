## 2. 模块概述

### 2.1 职责

配置管理模块负责应用配置的存储、读取、更新和备份，包括用户配置、设备 Token、窗口设置等。

### 2.2 功能范围

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 配置读取 | 启动时加载配置 | P0 |
| 配置写入 | 变更时保存配置 | P0 |
| 配置验证 | 验证配置有效性 | P0 |
| 配置备份 | 定期备份配置 | P1 |
| 配置重置 | 恢复默认配置 | P1 |
| Token 加密 | 敏感信息加密存储 | P0 |

### 4.2 核心类说明

#### 4.2.1 ConfigManager

配置管理器，单例模式。

```typescript
class ConfigManager {
  private static instance: ConfigManager | null = null;
  private configPath: string;
  private config: AppConfig | null = null;
  private crypto: CryptoService;
  
  private constructor() {
    this.configPath = path.join(app.getPath('userData'), 'config.json');
    this.crypto = new CryptoService();
  }
  
  static getInstance(): ConfigManager {
    if (!ConfigManager.instance) {
      ConfigManager.instance = new ConfigManager();
    }
    return ConfigManager.instance;
  }
  
  async load(): Promise<AppConfig> {
    try {
      const data = await fs.readFile(this.configPath, 'utf-8');
      const config = JSON.parse(data) as AppConfig;
      
      // 解密敏感信息
      if (config.miHome.token) {
        config.miHome.token = this.crypto.decrypt(config.miHome.token);
      }
      if (config.openclaw.token) {
        config.openclaw.token = this.crypto.decrypt(config.openclaw.token);
      }
      
      this.config = config;
      return config;
    } catch (error) {
      // 文件不存在或损坏，返回默认配置
      this.config = this.getDefaultConfig();
      await this.save();
      return this.config;
    }
  }
  
  async save(config?: Partial<AppConfig>): Promise<void> {
    if (config) {
      this.config = { ...this.config!, ...config };
    }
    
    // 加密敏感信息
    const configToSave = { ...this.config! };
    if (configToSave.miHome.token) {
      configToSave.miHome.token = this.crypto.encrypt(configToSave.miHome.token);
    }
    if (configToSave.openclaw.token) {
      configToSave.openclaw.token = this.crypto.encrypt(configToSave.openclaw.token);
    }
    
    configToSave.updatedAt = new Date();
    
    await fs.writeFile(
      this.configPath,
      JSON.stringify(configToSave, null, 2),
      'utf-8'
    );
  }
  
  get<T>(key: string): T {
    const keys = key.split('.');
    let value: any = this.config;
    for (const k of keys) {
      value = value?.[k];
    }
    return value as T;
  }
  
  async set<T>(key: string, value: T): Promise<void> {
    const keys = key.split('.');
    const lastKey = keys.pop()!;
    
    let current: any = this.config;
    for (const k of keys) {
      current = current?.[k];
    }
    
    if (current) {
      current[lastKey] = value;
      await this.save();
    }
  }
  
  async reset(): Promise<void> {
    this.config = this.getDefaultConfig();
    await this.save();
  }
  
  private getDefaultConfig(): AppConfig {
    return {
      version: '1.0',
      userId: generateUserId(),
      miHome: {
        region: 'cn',
      },
      window: {
        width: 400,
        height: 600,
        alwaysOnTop: true,
        opacity: 0.9,
        skipTaskbar: true,
      },
      appearance: {
        theme: 'system',
        fontSize: 14,
        language: 'zh-CN',
      },
      openclaw: {
        enabled: false,
        url: 'http://localhost:8000',
        autoConnect: true,
      },
      batchScripts: {
        rootDir: path.join(homedir(), 'mijia-scripts'),
        scripts: [],
      },
      devices: {
        autoRefresh: true,
        refreshInterval: 300,
      },
      logging: {
        level: 'info',
        maxFiles: 7,
        maxSize: '10m',
      },
      createdAt: new Date(),
      updatedAt: new Date(),
    };
  }
}
```

#### 4.2.2 CryptoService

加密服务类。

```typescript
class CryptoService {
  private algorithm = 'aes-256-cbc';
  private key: Buffer;
  private iv: Buffer;
  
  constructor() {
    // 使用机器唯一标识生成密钥
    const machineId = getMachineId();
    this.key = crypto.createHash('sha256').update(machineId).digest();
    this.iv = crypto.createHash('md5').update(machineId).digest();
  }
  
  encrypt(plainText: string): string {
    const cipher = crypto.createCipheriv(this.algorithm, this.key, this.iv);
    let encrypted = cipher.update(plainText, 'utf8', 'hex');
    encrypted += cipher.final('hex');
    return encrypted;
  }
  
  decrypt(cipherText: string): string {
    const decipher = crypto.createDecipheriv(this.algorithm, this.key, this.iv);
    let decrypted = decipher.update(cipherText, 'hex', 'utf8');
    decrypted += decipher.final('utf8');
    return decrypted;
  }
  
  hash(data: string): string {
    return crypto.createHash('sha256').update(data).digest('hex');
  }
}
```

## 8. 安全设计

### 8.1 Token 加密

使用 AES-256-CBC 算法加密敏感信息：

```typescript
// 加密流程
plainText (Token) 
    │
    ▼
AES-256-CBC Encrypt (key + iv)
    │
    ▼
cipherText (存储到配置文件)
```

### 8.2 密钥管理

```typescript
// 密钥派生
function deriveKey(): Buffer {
  // 使用机器唯一标识
  const machineId = getMachineId();  // CPU ID + MAC 地址 + 磁盘序列号
  return crypto.createHash('sha256').update(machineId).digest();
}

// 机器 ID 获取
function getMachineId(): string {
  const cpus = os.cpus()[0];
  const mac = getMacAddress();
  const disk = getDiskSerial();
  return `${cpus.model}-${mac}-${disk}`;
}
```

## 9. 异常处理

### 9.1 异常类型

| 异常 | 触发条件 | 处理方式 |
|------|----------|----------|
| ConfigNotFound | 配置文件不存在 | 创建默认配置 |
| ConfigCorrupted | 配置文件损坏 | 尝试恢复备份，或重置 |
| DecryptFailed | 解密失败 | 提示用户重新登录 |
| WriteFailed | 写入失败 | 重试 3 次，失败后提示 |

## 10. 备份策略

### 10.1 自动备份

```typescript
// 每次配置更新前自动备份
async function saveWithBackup(config: Partial<AppConfig>): Promise<void> {
  // 创建备份
  const timestamp = new Date().toISOString().split('T')[0];
  const backupPath = path.join(
    app.getPath('userData'),
    'backups',
    `config.${timestamp}.json`
  );
  
  await fs.copyFile(configPath, backupPath);
  
  // 保存新配置
  await configManager.save(config);
  
  // 清理旧备份 (保留最近 7 天)
  await cleanupOldBackups(7);
}
```

### 10.2 手动备份

```typescript
async function backup(): Promise<string> {
  const timestamp = new Date().toISOString().replace(/[:.]/g, '-');
  const backupPath = path.join(
    app.getPath('userData'),
    'backups',
    `config.manual.${timestamp}.json`
  );
  
  await fs.copyFile(configPath, backupPath);
  return backupPath;
}

async function restore(backupPath: string): Promise<void> {
  await fs.copyFile(backupPath, configPath);
  await configManager.load();  // 重新加载
}
```
