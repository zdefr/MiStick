## 当前设计关注点

| 模块 | 关注点 |
|------|--------|
| M01 | 扫码登录状态机、会话复用、云端同步失败、缓存回退 |
| M02 | 云控能力边界、设备不可控、控制服务可用性 |
| M03 | 扫码登录弹窗、禁用态渲染、窗口行为 |
| M04 | 认证路径元数据、兼容字段加密、配置校验、备份恢复、权限处理 |

## 快速开始

```bash
# 前端与主进程测试
npm run test

# Python 控制服务测试
cd python-service
pytest

# 生成覆盖率报告
npm run test:coverage
```
